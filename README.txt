TuftyShop - Sito vetrina (statico)
Files inclusi:
- index.html  -> pagina principale
- styles.css  -> foglio di stile

Istruzioni rapide per pubblicare gratis (2 opzioni):

A) GitHub Pages (consigliato per semplicità)
1. Crea un account su https://github.com (se non ce l'hai).
2. Crea un nuovo repository pubblico, ad esempio "tuftyshop".
3. Carica i file index.html e styles.css nella root del repo.
4. Vai in Settings -> Pages, scegli branch 'main' e root '/', salva.
5. Dopo pochi minuti il sito sarà disponibile all'indirizzo: https://<tuo-username>.github.io/tuftyshop/

B) Netlify (più flessibile, drag & drop)
1. Vai su https://www.netlify.com e crea un account (puoi collegarlo a GitHub).
2. Nella dashboard clicca "New site" -> "Deploy manually" -> drag & drop dei file index.html e styles.css (o carica la cartella).
3. Netlify genererà un dominio gratuito del tipo https://something.netlify.app

Note importanti:
- Le immagini nel file index.html usano link pubblici (Unsplash). Se vuoi sostituirle con foto tue, metti i file nella cartella 'images' e cambia gli src nelle <img>.
- Per usare un dominio personalizzato (es: tuftyshop.it) dovrai acquistarlo e collegarlo a GitHub Pages o Netlify (opzione a pagamento per dominio).
- Se vuoi che io personalizzi testi, colori, o aggiunga altre pagine, dimmelo e lo faccio io: posso aggiornare i file e inviarti un nuovo zip.

Buon lavoro!
